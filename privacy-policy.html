Privacy Policy for Cashat

Last updated: April 27, 2026

Cashat is a personal finance and transaction analysis app designed to help users organize and analyze VF-Cash transaction messages.

Data We Access

Cashat may request access to SMS messages only to detect and analyze VF-Cash transaction messages.

How We Use Data

SMS data is used to extract transaction details such as amount, date, transaction type, and related phone number when available.

Data Storage

Transaction data may be stored locally on the user's device and/or in Firebase services if cloud features are enabled.

Data Sharing

Cashat does not sell user data and does not share SMS content with advertisers or third parties.

Analytics

Cashat may use Firebase Analytics to understand general app usage, crashes, and performance. Analytics data does not include full SMS message content.

User Control

Users can revoke SMS permissions at any time from Android settings or uninstall the app.
